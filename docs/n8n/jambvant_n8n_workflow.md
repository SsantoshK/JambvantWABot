# Jambvant n8n Workflow

This folder contains the exported n8n workflow and documentation for the Jambvant WhatsApp Audio ChatBot.

- `jambvant_n8n_workflow.json` — n8n export (workflow nodes + connections)
- `jambvant_n8n_workflow.md` — human-readable guide / node details

Notes:
- Import `jambvant_n8n_workflow.json` into n8n (Workflows → Import) and configure credentials for WhatsApp, Google, DeepL, OpenAI, and PostgreSQL.
- Ensure the environment variables in the workflow match your n8n credentials configuration.
